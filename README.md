## 开发指南
**1.准备工作**

 - 首先，申请账号：点击 https://mp.weixin.qq.com/wxopen/waregister?action=step1
   填写资料，就可以拥有自己的小程序帐号。
   
 - 登录 https://mp.weixin.qq.com ，在菜单 “设置”-“开发设置” 看到小程序的 AppID 
 - 下载微信web开发者工具


**2.开始旅程**

 - 克隆整个项目
 - 打开开发者工具，新建小程序项目，浏览到组队小管家目录下 填写你的AppID
